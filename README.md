## Quickstart

First, clone this repository. Then:

```bash
> bundle install
> bin/rails db:create
> bin/rails db:migrate
```

## Testing

To run the tests: `bundle exec rspec`


## To view in browser

1. Start the server `bin/rails server`
2. Navigate to localhost:3000
